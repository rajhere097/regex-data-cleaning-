# Regex Data Cleaning & Feature Engineering

## 📌 Overview
This project demonstrates real-world data cleaning and text processing using Python and Regular Expressions (regex). It covers practical scenarios such as column normalization, log parsing, and NLP-style text preprocessing.

## 🚀 Key Features
- Clean and standardize messy column names
- Remove special characters and normalize text
- Extract structured data from logs using regex
- Perform NLP preprocessing (tokenization, URL removal, etc.)
- Feature extraction from unstructured text data
- Handle real-world messy datasets (CSV files)

## 🛠️ Tools & Technologies
- Python
- Pandas
- Regular Expressions (`re`)

## 📂 Project Workflow
1. Load raw data (CSV files)
2. Clean column names using regex patterns
3. Process text data (remove noise, normalize format)
4. Extract useful features using `findall()` and pattern matching
5. Save cleaned datasets for further analysis

## 🧠 Key Concepts Used
- `re.sub()` → replace patterns
- `re.findall()` → extract data
- Regex chaining (multiple patterns)
- Data cleaning pipelines
- Feature engineering from text

## 📊 Use Cases
- ETL pipelines
- Log file analysis
- Text preprocessing for Machine Learning
- Data cleaning for analytics projects

## 📎 Files Included
- `Regex.ipynb` → main notebook with all implementations
- Cleaned datasets (CSV)
